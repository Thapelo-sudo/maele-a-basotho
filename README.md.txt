# AI Proverbs Insights App

**AI-Powered Interpreter of Sesotho Proverbs (Maele)**  
A simple Streamlit app to explore Basotho proverbs, view Sesotho interpretations and English translations, and manage the dataset via Firebase.

---

## Features
- Search and browse Basotho proverbs.
- Display Sesotho interpretation and English translation.
- Admin interface to add, edit, or delete proverbs (password-protected).
- Data persisted in Firebase Firestore (optional).
